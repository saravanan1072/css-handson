what is css?

  1. css stands or Cascading Style Sheet

  2. CSS describe how HTML element are to be displayed on screen,paper,or in other media

  3. css is not neccesary CSS saves a lot of work .it can control the layout of multiple web pages all at once

  4. External stylesheets are stored in CSS files

CSS can be added to HTML document in 3 ways

  • Inline - by using the style attribute inside the elements
  • Internal-by using a style element in the header section
  • External-by using a link element to link an external CSS files.
  • the most common way to add CSS.is to keep the style in external CSS files.however, in this
    tutorial we will use inline and internal styles     ,because this is easier to demonstrate, and easier
    for you to try it yourself.

When comparing CSS class vs ID the difference is that CSS class applies a style to multiple elements,ID,on the other hand,applies a style tp one unique element.ID is also
special in that you can use a special URL to link directly to an element and its used by JavaScript.
In CSS,selectors are used to target specific element or range of element on a web page.Once an element has been targeted,a style or set of styles can be applied to
element.
there is a wide range of selectors available.Two of the mostt commonly used are class and ID.Both are used to target element to which a style shold be applied.
